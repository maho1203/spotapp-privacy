# プライバシーポリシー

本アプリ（SpotApp）は、ユーザーのプライバシーを尊重し、適切に個人情報を取り扱います。

## 1. 収集する情報
本アプリでは、以下の情報を収集する場合があります。

- ID・名前（アプリログイン用としてユーザーが入力したもの）
- 位置情報（スポットを登録する際に使用）

## 2. 情報の利用目的
収集した情報は以下の目的で利用します。

- アプリ内でのスポット登録および管理機能の提供
- 利便性向上のためのアプリ機能の改善

## 3. 情報の第三者提供
ユーザー情報を第三者へ提供することはありません。

ただし、法令で要求された場合にはこの限りではありません。

## 4. 使用している外部サービス
本アプリでは以下の外部サービスを利用しています。

- Firebase（データ保存のため）
- Google Maps API（地図表示のため）

各サービスのプライバシーポリシーはそれぞれをご確認ください。
